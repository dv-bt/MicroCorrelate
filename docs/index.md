# MicroCorrelate

A simple Python library for handling correlative microscopy datasets.

```{toctree}
:maxdepth: 1
:hidden:

api
```

## Modules

- **core** — intensity-based registration and correlation workflows
- **io** — reading and writing image files (TIFF, HDF5, OME-Zarr)
- **stitching** — image stitching utilities
- **napari** — napari-based interactive landmark-based registration
- **callbacks** — progress and event callbacks
- **utils** — miscellaneous helpers

## Installation

```bash
pip install microcorrelate
```
